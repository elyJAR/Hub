# Changelog

All notable changes to Hub will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial MVP release
- User discovery and presence system
- Connection request/accept mechanism
- Real-time text messaging
- Typing indicators
- WebSocket-based communication
- JWT session authentication
- Rate limiting per session
- Responsive UI for desktop and mobile
- Avatar selection system
- Docker support
- Comprehensive documentation

### Features
- Zero-installation browser-based platform
- Local network only (no internet required)
- Real-time user presence updates
- Connection request system with 30s timeout
- Text messaging with delivery status
- Typing indicators
- Session management with auto-cleanup
- Rate limiting to prevent spam
- Responsive design for all screen sizes

### Technical
- Next.js 14 with App Router
- TypeScript throughout
- Custom WebSocket server
- Tailwind CSS for styling
- Zod for message validation
- JWT for session tokens
- Docker containerization
- CI/CD with GitHub Actions
