## Development Commands

```bash
# Install dependencies
npm install

# Run development server
node node_modules/tsx/dist/cli.mjs server.ts

# Run tests
npm test

# Run linter
npm run lint

# Format code
npm run format

# Build for production
npm run build

# Start production server
npm start

# Run with Docker
docker-compose up
```
