# Public Assets

This folder contains static assets served by the application.

## Required Icon Files

To complete the PWA setup, add the following icon files:

- **icon-192x192.png** - 192x192 pixels (PWA icon)
- **icon-512x512.png** - 512x512 pixels (PWA icon)
- **apple-touch-icon.png** - 180x180 pixels (iOS home screen icon)
- **favicon.ico** - 32x32 pixels (browser tab icon)

## Generating Icons

You can use these tools to generate all required icons from a single source image:

- [Real Favicon Generator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)

## Files

- `manifest.json` - PWA manifest file for installable web app
- `.gitkeep-icons` - Reminder to add icon files
